# v2.0.0

重构：

- 增加项目总览界面：支持四种视图模式，甘特图、看板、列表、时间轴
- 重构原项目界面，只支持看板模式与列表模式
- 重构时间轴/甘特图页面：
  - 按照项目进行分组，每个项目分组下按照一级任务的日期设置显示对应任务条
  - 一级任务支持展开子任务，显示所有下一级且设置了日期的子任务
  - 底部按照项目进行分组，显示每个项目下未设置日期的任务，保持任务树状关系显示，如果子任务未设置日期，需要显示对应的上级任务
- 重构日历页面：
  - 按照项目进行分组，每个项目分组下按照一级任务的日期设置显示对应任务条
  - 一级任务支持展开子任务，显示所有下一级且设置了日期的子任务
  - 日历页面不显示未设置日期的任务

优化：

- 减少任务详情卡片 header 的高度，与主内容区 header 高度保持一致
- 优化下拉选择框宽度，过长文本不换行，自动增加宽度
- 优化 markdown 编辑器 blockquote 样式， app 端支持插入本地文件引用
- sidebar 中增加横线分割线显示，参考样式 --------收起/展开 icon----------
  - sidebar 顶部固定五个视图，支持收起/展开，收起时显示顶部三个按钮，展开显示全部五个按钮
  - sidebar 中间的项目列表，当项目超过三个时，显示展开收起分割线，展开状态显示全部项目，收起后只显示前三个项目
  - sidebar 底部的标签列表，当标签超过三个时，显示展开收起分割线，展开状态显示全部标签，收起后只显示前三个标签
- 增加全局 tooltip 组件，点击非 tooltip 区域时，tooltip 自动隐藏；鼠标移开目标元素时，tooltip 自动隐藏
- sidebar 收起后，支持鼠标划过显示 tooltip，显示对应名称
- sidebar 收起后，上面新增的横线分割线只显示短横线，不显示收起/展开 icon
- 暗色模式下，优化文字颜色，不显示纯白色文字，多选框显示为暗色模式
- 日历页面的日历表头国际化、“today”按钮国际化
- 任务详情卡片中，如果是新建任务，且没有设置任务标题、项目、board 看板时，不显示底部子任务相关内容
- 优先级增加 紧急 状态 和可忽略 状态
- 任务详情卡片中，调整标签选择：缩小标签选择按钮，标签选择按钮适中固定在最前面
